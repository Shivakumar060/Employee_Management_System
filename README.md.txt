# Employee Management System

This is a web-based Employee Management System built using Flask. 
It helps manage employee records, track attendance, and handle leave requests efficiently.

Features
- User login and authentication
- Add, update, delete employees
- Attendance tracking
- Leave management
- Activity logs
- Export data (CSV/PDF)

Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite Database

Project Structure
- app.py → Main backend
- templates/ → HTML files
- static/ → CSS & JS
- employees.db → Database

How to Run
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the application:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
