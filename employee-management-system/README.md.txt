# Employee Management System

## 📌 Overview
A web-based Employee Management System built using Flask. It helps manage employee records, attendance, and leave requests.

## 🚀 Features
- User authentication (login/logout)
- Add, update, delete employees
- Attendance tracking
- Leave management
- Activity logs

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS, JavaScript
- SQLite

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python app.py
3. Open:
   http://127.0.0.1:5000/
